# 시트 2: 📊 Executive Dashboard (L1 부문장 뷰)

## 목적
모든 KPI의 현황을 한눈에 파악. 수식 전용 시트 — 직접 입력 없음.

---

## 레이아웃 전체 구조

```
행 1-2   : 헤더 (제목, 기준월 표시)
행 3-4   : 빈 행 (여백)
행 5-11  : [섹션 1] KPI 카드 7개 (RAG 상태 + 달성률)
행 12-13 : 빈 행
행 14-22 : [섹션 2] 분기 타임라인 Progress
행 23-24 : 빈 행
행 25-30 : [섹션 3] SNS 채널 트렌드 요약
행 31-32 : 빈 행
행 33-38 : [섹션 4] 알림 패널 (Amber/Red 항목 자동 표시)
```

### 섹션 1: KPI 카드 (행 5-11)

각 KPI마다 3열 × 1행 구조로 카드 배치. 총 4카드/행 × 2행.

#### 카드 구조 (KPI_1A 예시: B5:D7)

| 셀 | 내용 | 수식 |
|----|------|------|
| B5 | `60주년 사사 편찬` | 텍스트, 굵게, 배경 #37474F, 흰색 글자 |
| B6 | RAG 상태 아이콘 | `=IF(C11>=90,"🟢",IF(C11>=70,"🟡","🔴"))` |
| C6 | 달성률 | `='✅ 마일스톤_분기입력'!D5` |
| D6 | `비중 15%` | 텍스트 |
| B7 | 최근 마일스톤 | `='✅ 마일스톤_분기입력'!E5` |

#### 전체 카드 배치표

| 위치 | KPI | 비중 | 달성률 참조 셀 |
|------|-----|------|---------------|
| B5:D7 | 60주년 사사 편찬 (KPI_1A) | 15% | `='✅ 마일스톤_분기입력'!B20` |
| F5:H7 | JV Case Study (KPI_1B) | 10% | `='✅ 마일스톤_분기입력'!B30` |
| J5:L7 | 홈페이지 고도화 (KPI_2) | 20% | `='✅ 마일스톤_분기입력'!B40` |
| B9:D11 | External Comm. (KPI_3) | 15% | `='📱 SNS_주간입력'!SUMMARY_RAG` |
| F9:H11 | DAX 절감 (KPI_4) | 10% | `='🤖 DAX_월별입력'!B_YTD_RATE` |
| J9:L11 | Commercial PR (KPI_5) | 15% | `='📰 홍보자료_건수입력'!ACHIEVE_RATE` |

#### 달성률 → RAG 조건부 서식 수식

적용 범위: 각 카드의 달성률 셀
```
규칙 1 (Green):  =값>=90  → 배경 #C8E6C9, 글자 #1B5E20
규칙 2 (Amber):  =AND(값>=70,값<90) → 배경 #FFF9C4, 글자 #F57F17
규칙 3 (Red):    =값<70   → 배경 #FFCDD2, 글자 #B71C1C
```

### 섹션 4: 알림 패널 (행 33-38)

자동으로 Amber/Red KPI만 표시되는 알림 목록.

| 셀 | 수식 |
|----|------|
| A33 | `⚠️ 주의 필요 항목` (헤더) |
| A34 | `=IFERROR(INDEX(알림목록,MATCH("🔴",알림_RAG,0)),"이슈 없음")` |
| A35 | `=IFERROR(INDEX(알림목록,MATCH("🟡",알림_RAG,0)),"")` |

**알림목록 Named Range 정의:**
```
알림목록  = {KPI명 배열}
알림_RAG  = {각 KPI RAG 배열}
```

구체적 배열 수식 (A34에 입력):
```
=ARRAYFORMULA(
  IFERROR(
    FILTER(
      {"60주년사사";"JV케이스";"홈페이지";"External Comm";"DAX";"Commercial PR";"Internal Comm";"CSR"},
      {마일스톤!B20;마일스톤!B30;마일스톤!B40;SNS!SUMMARY;DAX!YTD_RATE;홍보!RATE;정량!INTERNAL;마일스톤!CSR_RATE}<70
    ),
    "이슈 없음"
  )
)
```
