# 시트 4: ✅ 마일스톤_분기입력

## 목적
정성 KPI (사사, JV, 홈페이지, 외부수상, CSR)의 마일스톤 달성 여부를 체크박스로 관리.
배점 가중 평균으로 달성률(%) 자동 산출.

---

## 전체 구조

```
행 1-2   : 헤더
행 3-20  : KPI_1A 60주년 사사 편찬
행 22-37 : KPI_1B JV Case Study
행 39-54 : KPI_2  홈페이지 고도화
행 56-65 : KPI_5B 외부 수상
행 67-78 : KPI_7  CSR Milestone
행 80-85 : 종합 달성률 요약
```

## 체크박스 삽입 방법

Google Sheets: 셀 선택 → 삽입 → 체크박스
- TRUE = 달성, FALSE = 미달성
- E열 수식: `=IF(C열=TRUE, D열, 0)`
