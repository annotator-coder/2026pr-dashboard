| 1Q | 20점 | 기획사 선정 완료 | 기획사와 계약서 날인 완료 | 계약 협상 중(RFP 발송)→10점 | 계약서 사본 |

| 3Q | 25점 | 테마사 착수 + DAX 확장 콘텐츠 | 테마사 집필 계약 + DAX 콘텐츠 1건 이상 시안 | 테마사만 착수→15점 / DAX만→10점 | 계약서, 시안 파일 |
| 4Q | 30점 | 초고 1차 감수 + Archive 착수 | 감수 완료 보고서 제출 + Archive 개발사 계약 | 감수만→20점 / Archive만→10점 | 감수 보고서, 계약서 |

**검증 주체**: 홍보부문장 (분기 리뷰 시 확인)

**가점 규정**: 초고 완성도가 감수위원 평균 평점 4.0/5.0 이상 시 +3점 (최대 103점)

## KPI 2. 홈페이지 고도화 (비중 20%)

### 마일스톤별 완료 기준

| 분기 | 배점 | 마일스톤 | 완료 기준 | 부분 점수 규칙 | 증빙 자료 |
|------|------|----------|----------|--------------|----------|
| 1Q | 10점 | 기획 착수 | 사이트맵 초안 + 벤치마킹 보고서 부문장 보고 완료 | 둘 중 하나만→5점 | 보고 자료 |

- 콘텐츠 진도율 = 완료 페이지 수 / 전체 목표 페이지 수 × 100

## KPI 6. 구성원 Internal Comm 응답률

### 서베이 설계 기준

| 항목 | 기준 |
|------|------|
| 측정 시점 | 2Q 말 (6월 중순 중간 점검) + 4Q 말 (11~12월 최종 측정) |
| 조사 대상 | 전체 임직원 (홍보부문 제외) |
| 최소 응답 수 | 전체 임직원의 30% 이상 (미달 시 결과 무효) |
| 조사 방법 | 사내 인트라넷 설문 (익명) |
| 문항 수 | 5~7문항 (리커트 5점 척도) |
| 긍정 응답 정의 | 5점 척도 기준 4점 또는 5점 응답 비율 |

### 서베이 문항 (안)

| 번호 | 문항 | 배점 가중치 |
|------|------|-----------|
| Q1 | 나는 회사의 2026년 전략 방향과 주요 현황을 잘 이해하고 있다 | 30% |
| Q2 | 나는 회사의 미래 투자 및 혁신 활동(DAX 등)에 대해 충분히 알고 있다 | 25% |
| Q3 | 나는 회사 구성원으로서 자부심을 느낀다 | 20% |
| Q4 | 나는 회사의 비전과 방향성에 공감한다 | 15% |
| Q5 | 사내 커뮤니케이션 채널(뉴스레터, 영상, 행사 등)이 유익하다 | 10% |

**긍정 응답률 산출 수식**:
```
긍정 응답률 = Σ(각 문항 4·5점 응답 수 × 가중치) / 전체 응답 수 × 100
```

**목표 80% 설정 근거**:
- 글로벌 에너지 기업 Internal Comm 벤치마크 평균: 75~82%
- '위버멘쉬' 테마 신년사 이후 구성원 관심도 상승 기대
- 60주년 행사 연계 긍정 효과 반영
