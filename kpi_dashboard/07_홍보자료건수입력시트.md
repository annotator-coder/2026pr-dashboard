# 시트 7: 📰 홍보자료_건수입력

## 목적
전략사업 홍보자료 건수(KPI_5A, 목표 22건)를 건별로 기록하고 달성률 추적.

---

## 레이아웃

```
행 1-2   : 헤더 + 목표 요약
행 3-4   : 열 헤더
행 5-54  : 홍보자료 건별 입력 (최대 50건)
행 55-65 : 분기별 집계 및 달성률
행 66-75 : KPI 달성률 요약
```

---

## 목표 요약 (행 1-2)

| 셀 | 내용 | 값 |
|----|------|----|
| A1 | `전략사업 홍보자료 건수 목표` | |
| B1 | `=⚙️설정!D23` | 22건 |
| C1 | `('25년 실적:` | |
| D1 | `=⚙️설정!C23&"건)")` | 19건 |

## 열 헤더 (행 3-4)

| 열 | A | B | C | D | E | F | G |
|----|---|---|---|---|---|---|---|
| 3  | NO | 발표일 | 제목 | 분류 | 주요 내용 (한줄) | 분기 | 매체 수 |
| 4  | | 🟦 | 🟦 | 🟦 드롭다운 | 🟦 | 🟩 자동 | 🟦 |

### D열 분류 드롭다운 옵션

데이터 유효성 검사 → 목록:
```
BC(Business Council)
DAX
신에너지
기존사업
ESG/탄소중립
기타 전략사업
```

### F열 분기 자동 계산 수식 (F5 예시)

```
=IF(B5="","",
  IF(MONTH(B5)<=3,"1Q",
    IF(MONTH(B5)<=6,"2Q",
      IF(MONTH(B5)<=9,"3Q","4Q"))))
```

## 건별 입력 행 (행 5-54)

| 행 | A | B | C | D | E | F | G |
|----|---|---|---|---|---|---|---|
| 5  | 1 | 🟦 날짜 | 🟦 제목 | 🟦 분류 | 🟦 내용 | `수식` | 🟦 |
| 6  | 2 | 🟦 | 🟦 | 🟦 | 🟦 | `수식` | 🟦 |

**A열 자동 번호 수식 (A5):**
```
=IF(B5="","",COUNTA($B$5:B5))
```

## 분기별 집계 (행 55-65)

| 행 | A | B (건수) | C (누계) | D (분기 목표) | E (달성률) |
|----|---|---------|---------|------------|----------|
| 55 | **분기별 현황** | | | | |
| 56 | 1Q | `=COUNTIF(F5:F54,"1Q")` | `=B56` | 5 | `=B56/D56*100` |
| 57 | 2Q | `=COUNTIF(F5:F54,"2Q")` | `=B56+B57` | 4 | `=C57/(D56+D57)*100` |
| 58 | 3Q | `=COUNTIF(F5:F54,"3Q")` | `=C57+B58` | 6 | `=C58/SUM(D56:D58)*100` |
| 59 | 4Q | `=COUNTIF(F5:F54,"4Q")` | `=C58+B59` | 7 | `=C59/22*100` |
| 60 | **연간 합계** | `=SUM(B56:B59)` | | 22 | `=B60/22*100` |

> 분기별 목표 배분: 1Q=5, 2Q=4, 3Q=6, 4Q=7 (전략 집중도에 따라 조정 가능)

## KPI 달성률 요약 (행 72-78)

| 행 | A | B | C |
|----|---|---|---|
| 72 | **KPI_5A 달성률 요약** | | |
| 73 | YTD 누계 건수 | `=B60` | 건 |
| 74 | 연간 목표 건수 | 22 | 건 |
| 75 | 달성률 (%) | `=B73/22*100` | % |
| 76 | RAG | `=IF(B75>=90,"🟢",IF(B75>=70,"🟡","🔴"))` | |
| 77 | 전년 대비 증감 | `=B73-19` | `=IF(B77>=0,"▲"&B77&"건","▼"&ABS(B77)&"건")` |

**Named Range:**
- `PR_COUNT_YTD` = B73
- `PR_ACHIEVE_RATE` = B75

---

## 입력 운영 방법

1. 홍보자료 배포 후 당일 또는 익일 B~G열 입력
2. 제목(C열): 보도자료 제목 그대로 기재
3. 분류(D열): 드롭다운에서 선택 (복수 해당 시 주된 항목 선택)
4. 매체 수(G열): 해당 자료 기반 기사화된 매체 수 (배포 후 1주 시점 기준)

## 스프레드시트 필터 설정

- 행 3 고정 (헤더 고정): 보기 → 고정 → 3행까지
- D열 (분류) 필터 활성화: 데이터 → 필터 생성
- F열 (분기) 기준 정렬 기본 설정
