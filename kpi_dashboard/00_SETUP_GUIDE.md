# GS칼텍스 홍보부문 KPI Dashboard — Google Sheets 설정 가이드

## 시트 구성 (탭 순서)

| 탭 번호 | 시트명 | 용도 | 갱신 주기 |
|---|---|---|---|
| 1 | `⚙️ 설정` | 목표치·가중치 마스터 | 연 1회 |
| 2 | `📊 Executive Dashboard` | L1 부문장 뷰 | 자동 (참조) |
| 3 | `📋 정량_월별입력` | 수치 KPI 데이터 입력 | 월 1회 |
| 4 | `✅ 마일스톤_분기입력` | 정성 KPI 체크리스트 | 분기 1회 |
| 5 | `📱 SNS_주간입력` | SNS 채널별 주간 실적 | 주 1회 |
| 6 | `🤖 DAX_월별입력` | 업무 절감 시간 상세 | 월 1회 |
| 7 | `📰 홍보자료_건수입력` | 홍보자료 건수 상세 | 수시 |

## Google Sheets 생성 순서

1. 새 스프레드시트 생성
2. 탭 이름을 위 순서대로 변경
3. 각 시트별 구조 파일(01~07)의 내용 참고하여 셀 구성
4. `⚙️ 설정` 시트 목표치 먼저 입력
5. Named Range 설정 (각 시트 가이드 참고)

## 색상 컨벤션 (전체 공통)

| 셀 색상 | 의미 |
|---|---|
| 🟦 연파랑 (#E3F2FD) | 입력 셀 (사용자가 직접 입력) |
| 🟩 연초록 (#E8F5E9) | 수식 셀 (자동 계산, 수정 금지) |
| 🟨 연노랑 (#FFFDE7) | 헤더/레이블 |
| ⬜ 흰색 | 참조/설명 영역 |

## RAG 조건부 서식 공통 규칙

- 🟢 Green : 달성률 ≥ 90%  → 배경 #C8E6C9, 글자 #1B5E20
- 🟡 Amber : 달성률 70~89% → 배경 #FFF9C4, 글자 #F57F17
- 🔴 Red   : 달성률 < 70%  → 배경 #FFCDD2, 글자 #B71C1C
